Documentation

****************
Installation
****************

::

    $ pip install -r requirements/docs.txt

****************
Run Server
****************

::

    $ cd docs/
    $ make html
    $ sphinx-autobuild ./ ./_build/html -H 0.0.0.0

Open the link ``http://localhost:8000`` on a browser.
